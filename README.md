# Stocks Explore PWA

This is a responsive stock-dashboard demo inspired by the supplied screenshots.

## Run locally

PWA features require HTTP/HTTPS, not opening `index.html` directly.

### Option 1: VS Code
Install the **Live Server** extension and open `index.html` with Live Server.

### Option 2: Python
```bash
python -m http.server 8080
```
Then open:
`http://localhost:8080`

## Install as a web app

### Android / Chrome
Open the website, tap the browser menu, then choose **Install app** or **Add to Home screen**.

### iPhone / Safari
Tap **Share**, then **Add to Home Screen**.

### Desktop Chrome / Edge
Click the install icon in the address bar.

## Deploy free

You can deploy the folder free using:
- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel

## Important
The project uses original demo branding and placeholder market data. Before production use, connect it to a licensed market-data API and add authentication.
